# Flutter with Firebase.
A mobile application with clean UI design. Full authentication support using Firesbase and FireStore. Suitable for both ios and android.

## Tech Stack

**Dart, Flutter, Firebase Authentication, Firestore**

## Running

One can run this project locally for any of these platforms. For desktop platforms,
please see the [Flutter docs](https://docs.flutter.dev/desktop) for the latest
requirements.

```bash
cd gallery/
flutter pub get
flutter run
```
